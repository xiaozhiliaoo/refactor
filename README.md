## How to use it

The **@Refactor** library is deployed to maven central. You can simply add it as a dependency:

### Apache Maven
```xml
<dependency>
    <groupId>com.github.xiaozhiliaoo</groupId>
    <artifactId>refactor</artifactId>
    <version>0.0.1</version>
</dependency>
```
